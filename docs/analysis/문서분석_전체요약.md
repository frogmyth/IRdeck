## 0. 핵심 비즈니스 정의 (CEO 확인)

> **AIsirius = 리테일 전문 AI 기업**
> AI가 핵심이고, 리테일 미디어 플랫폼은 AI를 적용하는 무대이다.

### 한 문장 정의
**리테일에 특화된 AI 기술**로 오프라인 매장을 지능형 미디어 공간으로 전환하는 기업.
종이/EPD(전자잉크)를 **LCD(안드로이드 탑재 기기)**로 대체하고, AI가 콘텐츠 생성, 매장 분석, 진열 최적화를 자율 수행하는 **AI 기반 리테일 미디어 플랫폼(RMN+ESL 통합)** 제공.

### 왜 "리테일 전문 AI"인가?
- 범용 AI가 아닌 **리테일 도메인에 깊이 특화된 AI**
- POS/ERP 데이터와 매장 센서를 결합한 **물리 매장 전용 AI 인텔리전스**
- 이미지/동영상 자동 생성 → 매장 콘텐츠에 최적화된 생성형 AI
- 플래노그램 분석 + AI 제안 → 진열 최적화, 가격 전략, 재고 관리 자동화
- On-Device AI → 매장 엣지에서 실시간 처리 (클라우드 의존도 최소화)

## 2. 핵심 소구 포인트 (중복 제거 후 정리)

### A. 시장 기회 (Marketing Opportunity)

#### A-1. DX에서 AX로의 패러다임 전환
- "DX가 자동화(Automation)라면, AX는 자율화(Autonomy)" - 사티아 나델라
- 2026년 현재 비즈니스 패러다임이 DX → AX(AI Transformation)로 급변
- ISM(In Store Merchandising) → AI ISO4OM(In-store O4O Merchandising)으로 시장 형성 변화

#### A-2. AX 시대의 4대 비즈니스 기회
1. **에이전틱 서비스/솔루션** (Agentic SaaS): 매장 내 고객 유형분석, 행동분석, 상품분석, 재고/선반관리
2. **산업 특화 AI** (Vertical AX): 범용 LLM → 리테일 특화 SLM 수요 급증
3. **AI 거버넌스/보안**: 온프레미스/프라이빗 AI 구축 → 데이터 주권 보장
4. **AI Build-in HW & Edge Computing**: On-Device AI & 3단 분산처리 시스템

#### A-3. 글로벌 시장 규모
- **ESL 시장**: 2024년 4조원, 연 20.8% 성장, 2030년 12조원 이상 예상
- **RMN & 사이니지 시장**: 2026~2030년 59조→97조원 규모
- **통합 시장 (RMN + ESL + 사이니지)**: 72조→97조원 (2026~2030)

#### A-4. 월마트 사례 (핵심 레퍼런스)
- 월마트 시총 1조 달러 돌파 (2026.02) - AI 투자가 주효
- 광고 사업이 매출보다 6배 빠르게 성장 (Walmart Connect 33% vs 소매 5%)
- 광고가 영업수익의 1/3 차지 → 고마진 광고가 저마진 소매를 보조
- 오프라인 매장 → 디지털 광고 미디어 플랫폼으로 변환 성공

### B. 비즈니스 모델 (Business Model)

#### B-1. AIsirius 핵심 정의
> AI 기반 ISO4OM Platform (In-store O4O Merchandising) 전문 기업
> 데이터(Cloud)와 지능(AI), Edge 접점(ESL/RMN/Multi-Modal)을 통합 네트워크로 연결하여
> 오프라인 매장의 DX를 넘어 AX를 주도

#### B-2. ISO4OM 3 Solution Integrated
1. **RMN Solution (Retail Media Network)**
   - 오프라인 매장을 가치 있는 광고 매체로 변환
   - 구매 고객의 동선/시선 분석 → 최적 타이밍 디지털 사이니지 광고 노출
   - 판매 수익 + 리테일 미디어 광고 수익 창출

2. **ESL Solution (Electro Self Label)**
   - 단순 가격 표시기 → 실시간 운영 최적화 도구
   - 다이나믹 프라이싱: 재고/시간에 따른 즉각 가격 업데이트
   - O4O 연동: QR → AI 에이전트 → 제품 상세정보/리뷰 즉시 제공

3. **Multi-Modal Solution (AI Vision, Voice & Sensors)**
   - 시각/음성/센서 데이터 결합 → 매장 입체 분석
   - 고객 행동/감정 분석 → 정교한 마케팅 데이터 생성
   - AI 에이전트 진열 실시간 모니터링 → 자율형 ISO4OM

#### B-3. HW 라인업
- Android Base IP Device Series (All Size)
- AI On-Device Series
- Multi-Modal 기능 HW (Vision, Voice & Sensors)

### C. CMS 기능 (Aisirius CMS 고유 소구점)

#### C-1. 기존 EPD 대비 차별화
- 단순 가격표 → 모든 스크린이 동적 광고판/세일즈 채널로 진화
- Traditional ESL → Aisirius Media Platform

#### C-2. 생성형 AI 콘텐츠 엔진
- 상품명/설명만으로 전문 스튜디오 수준 이미지/영상 자동 생성
- 4단계: 한국어 입력 → AI 프롬프트 엔지니어링 → 이미지/영상 생성 → 즉시 매장 적용
- 비규격 사이즈 맞춤 제작 (23, 29인치 와이드 등)
- 이벤트/시즌 배너, 고객 맞춤형 메시지 자동 생성

#### C-3. 프로모션 자동화
- N+N (1+1, 2+1), 구간별 할인, 정률/정액 할인
- 예약 스케줄링 (날짜, 시간, 분 단위)
- 카운트다운 표시 → 긴급성 부여 (홈쇼핑 "마감임박" 효과)

#### C-4. 매장 관리 시스템
- 디지털 트윈 기반 2D 시각적 매장 편집기
- PDA 현장 관리 (바코드 스캔 → 즉시 상품 교체, 품절 처리)
- 계층적 조직 구조 관리 (본사→국가→지점→층→구역→진열대)
- Excel 기반 대량 일괄 처리
- RBAC 권한 관리

#### C-5. 시스템 안정성
- On-Device Playback (서버 장애/네트워크 중단 시에도 정상 표시)
- Cloud / On-Premise 유연한 배포 옵션

#### C-6. 확장 비전
- 매장 벽을 넘어 고객 가정으로 (디지털 전단지, 타임세일 알림, 간편 주문)

### D. AI ISM 영업 특화 소구점 (SI 파트너용)

#### D-1. SI 파트너 핵심 가치
- **수익 모델 확장**: HW 단발성 → SW 라이선스 + 유지보수 Recurring
- **강력한 ROI 제시**: 인건비 절감 + 광고 수익 창출
- **고객 락인(Lock-in)**: 데이터 기반 관리 시스템 → 장기 파트너십

#### D-2. 6가지 핵심 기능 에코시스템
1. Pricing (Real-time): DB연동 실시간 가격/프로모션 관리
2. Touch (Info): 온디맨드 상세 정보 / 디지털 사양표
3. Promo (Gamification): 참여형 이벤트 → 체류시간 증대
4. Planning (Digital Twin): 매장 평면도 기반 직관적 관리
5. Pairing (Connectivity): 기기 간 연동 → 대화면 경험 확장
6. AI Gen (Content): 생성형 AI 마케팅 이미지/영상 제작

#### D-3. DB 동기화 방식의 우위
- 기존 이미지 전송 방식: 분 단위 반영, 오기입 위험, 고용량 부하
- AI ISM DB 동기화: 1초 이내 반영, ERP/POS 100% 일치, 초경량 텍스트

#### D-4. 게이미피케이션 프로모션
- 버블 터치 게임 → 쿠폰 획득 (QR 스마트폰 저장)
- 매장을 즐거운 공간으로 인식 → 체류시간/재방문율 증가

#### D-5. 리테일러 도입 ROI
- 운영 효율: 인건비/소모품 비용 90%↑ 절감

#### E-1. [출원 특허] 멀티모달 3단계 분산 AI 처리 시스템 (출원번호: 10-2025-0208402)

**정식 명칭**: 입력기반 계층분기 로직을 포함한 멀티모달 3단계 분산 인공지능 처리 시스템 및 그 방법

**핵심 구조: 3계층 자율 분기 AI**
```
제1계층: AI 엣지 디바이스 (사용자 최접점)
  ├── 멀티센서 입력 (카메라/마이크/근접/조도/온도)
  ├── 경량 AI 모델 (CNN/RNN) → 입력 난이도 판단
  ├── 저난이도: 즉시 처리 (가격 안내, 화면 활성화 등)

제2계층: AI 액세스 포인트 (중간 연산 노드)
  ├── 복수 엣지 이벤트 수집/분류
  ├── 지역 DB + 규칙 기반 로직 → 중간 판단
  ├── 처리 가능: 추천/비교/위치 안내 → 엣지로 반환
  └── 처리 불가: 서버로 요청 전송
        │
```

### 시연된 핵심 CMS 기능

#### H-1. 데이터 기반 콘텐츠 관리 (이미지 파일 방식 vs DB 방식)
- **기존 사이니지 문제**: 콘텐츠를 이미지/영상으로 만들어 전송 → 수정 시 이미지/영상 전부 재제작 필요
- **AIsirius 방식**: 모든 콘텐츠를 데이터화 → 데이터만 변경하면 디바이스에 즉시 반영
- **NoSQL DB**: 고정 형태가 아닌 유연한 데이터 구조 → 어떤 형태의 상품 데이터든 효율적 관리
- **논리적 태그**: 하나의 LCD 디바이스 내 각 영역이 고유 ID를 가진 개별 태그로 동작 (= 각각이 독립 ESL처럼 작동)

#### H-2. 실시간 태그/상품 관리
- 태그 삭제/추가/이동 → 디바이스 즉시 반영
- 상품 교체 → 데이터 변경만으로 디바이스 자동 업데이트
- 가격 변경 → 해당 상품을 사용하는 **모든 태그가 동시 업데이트**
- 즉시 적용 or **예약 스케줄링** (원하는 시간에 자동 변경)

#### H-3. 플래노그램 기반 매장 관리
- 국가 → 매장 → 층 → 구역 → 진열대 계층 구조
- 2D 평면도 기반 시각적 편집기 (층별 구역 생성, 크기 조절)
- 진열대 상세보기에서 디바이스 상태/콘텐츠 즉시 확인

#### H-4. 크로스 디바이스 연동 (Pairing)
- 23인치 태그 터치 → 연동된 29인치/32인치에 해당 상품 상세 정보 표시
- 작은 선반 디스플레이 → 큰 디스플레이로 확장 경험 제공

#### H-5. AI 이미지/동영상 생성 (자체 AI 서버)
- **AI 이미지**: 프롬프트 입력 → 프로모션 이미지 즉시 생성
  - 예: "HAPPY NEW YEAR, 일출, red horse, 따뜻한 톤" → 이벤트 이미지
  - 예: "SD 3D 피규어, 젊은 한국 남자, 아이돌 의상" → 캐릭터 이미지
- **AI 동영상**: AI 생성 이미지 → 7.5초 동영상으로 변환
  - 예: "잘 구워진 바게트, 나무 쟁반" 이미지 → 베이커리 동영상
- 생성된 콘텐츠를 CMS에서 즉시 상품에 적용 → 디바이스 자동 반영
- 자체 AI 서버 운영 (ComfyUI 기반), 지속적 학습/튜닝 중

#### H-6. 세부 스펙 표시 (가전 특화)
- 기존: EPD 태그 밑에 종이 세부스펙 프린트하여 부착
- AIsirius: LCD에서 노트북, 냉장고 등 세부스펙을 디지털로 표시 (종이 완전 대체)

#### H-7. 즉석 프로모션 관리
- 미리 계획하지 않은 가격 변경/세일 → 클릭 몇 번으로 즉시 처리
- 추가 증정 (1+1), 할인 등 프로모션 유형 다양
- 세일 중인 상품 필터링 → 일괄 관리 가능

#### H-8. 게이미피케이션 (방울 터뜨리기)

> **스토리라인: "리테일 전문 AI" → 이 AI가 적용되는 플랫폼 → 시장 기회 → 성과 → 투자**

### [Tier 1] AI 핵심 — 왜 우리가 "리테일 전문 AI"인가

1. **리테일 전문 AI 포지셔닝**: 범용 AI가 아닌 오프라인 매장에 특화된 도메인 AI
2. **AI 콘텐츠 자동 생성**: 상품명만으로 이미지/동영상 즉시 생성 → 매장 즉시 배포
3. **AI 매장 분석**: POS 데이터 연동 → 플래노그램 분석 → 진열/가격/재고 AI 제안
4. **AI 자율 운영**: 다이나믹 프라이싱, 프로모션 자동화, 실시간 센서 기반 의사결정
5. **On-Device AI**: 3단 분산처리 시스템 (특허), 매장 엣지에서 실시간 AI 처리

### [Tier 2] 플랫폼 — AI가 동작하는 무대

6. **리테일 미디어 플랫폼 (RMN+ESL 통합)**: 가격표시 + 정보 + 이미지 + 동영상 + 광고
7. **기존 기기 흡수력**: 매장의 기존 안드로이드 기기에 SW 설치만으로 생태계 편입
8. **CMS 통합 관리**: 다양한 사이니지/안드로이드 디바이스를 하나의 플랫폼에서 운영
9. **HW 유연성**: LCD 모든 사이즈 + stretched 비정형 + EPD → 중국 파트너사 소싱/합납품

### [Tier 3] 시장 기회 — 왜 지금인가

10. **DX→AX 전환**: 디지털화를 넘어 AI 자율화 시대 도래
11. **글로벌 시장 97조원**: RMN + ESL + 사이니지 통합 시장 (2030년)
12. **월마트 사례**: 광고 매출 6배 성장, 시총 1조 달러 → 오프라인 매장의 미디어화 검증

### [Tier 4] 전략 & 성과 — 실행력 증명

13. **글로벌 파트너십**: TGCS(도시바)/TCS(TATA) + 2028년 ESL 교체 주기 공략
14. **국내 트랙션**: 포토이즘 PoC, 회의실 안내 시스템 납품 완료, 대형마트 진입
15. **ROI 143%**: 도입 1년 내 투자비 전액 회수 (운영비 90%↑ 절감)

### [Tier 5] 비즈니스 모델 & 투자

16. **하이브리드 수익**: HW 판매 → SaaS 구독 → RMN 광고 Revenue Share (고마진 전환)
17. **J-Curve 로드맵**: 2026(35억) → 2028(247억) → 2030(1,100억/IPO)
18. **CEO 차별성**: ESL 2~4세대 세계 최초 개발 (삼성전기 출신)
19. **ESG**: Paperless + Food Waste Zero + 지역 상생
20. **확장 비전**: 매장→가정 (디지털 전단지, 타임세일 알림, 간편 주문)
