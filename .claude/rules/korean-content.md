# 한국어 콘텐츠 작성 규칙

## 기본 원칙
- 기본 언어: 한국어
- 기술 용어/고유명사: 영문 병기 (예: "전자잉크(EPD)")
- 회사명: "에이아이시리우스" 또는 "AIsirius" (문맥에 따라)
- 약어 첫 등장 시 풀네임 표기

## 톤 & 스타일
- **투자자 대상**: 전문적, 데이터 기반, 성장 가능성 강조
- **파트너 대상**: 협업 가치, 기술 시너지 강조
- **고객 대상**: 실용적 효과, ROI 중심

## 숫자/데이터 표기
- 금액: 억원/조원 단위 (예: "$2.34B → 약 3조원")
- 성장률: CAGR 표기 (예: "CAGR 13.9%")
- 연도: 4자리 (예: "2024년", "2030년")
- 퍼센트: % 기호 사용

## 키워드 통일
| 통일 용어 | 사용하지 않는 표현 |
|-----------|-------------------|
| 리테일 미디어 | 소매 미디어, 유통 미디어 |
| 전자잉크(EPD) | e-ink, 전자 종이 |
| 전자가격표시기(ESL) | 전자 라벨, e-label |
| 리테일 미디어 네트워크(RMN) | 소매 광고 네트워크 |
| 플래노그램 | 진열도, 선반 배치도 |
| 디지털 사이니지 | 전자 간판, 디지털 간판 |

## 문장 구조
- 슬라이드 제목: 명사형 종결 (예: "AI 기반 리테일 혁신")
- 불릿 포인트: 체언 종결 또는 간결한 서술 (예: "매출 23% 향상")
- 긴 설명: 경어체 사용 (예: "~합니다", "~입니다")
